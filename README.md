# Blinkit Data Analytics Dashboard 📊

## Project Overview
![Blinkit Dashboard](blinkit.png)

This project presents an interactive Power BI dashboard built using Blinkit grocery sales data. The dashboard helps analyze sales performance, customer behavior, outlet efficiency, and product trends through meaningful visualizations and KPIs.

## Objectives

- Analyze total sales and revenue performance
- Identify top-performing product categories
- Compare sales across outlet types and locations
- Evaluate customer ratings and purchasing patterns
- Monitor inventory and product distribution
- Support business decision-making through data insights

## Tools & Technologies

- Power BI
- Microsoft Excel / CSV Dataset
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling

## Key Performance Indicators (KPIs)

- Total Sales
- Average Sales
- Number of Items Sold
- Average Customer Rating
- Outlet Performance
- Category-wise Revenue
- Location-wise Sales Distribution

## Dashboard Features

### Sales Analysis
- Total Revenue Overview
- Monthly Sales Trends
- Category-wise Sales Performance

### Product Insights
- Best Selling Products
- Product Category Distribution
- Fat Content Analysis

### Outlet Analysis
- Outlet Size Comparison
- Outlet Type Performance
- Outlet Establishment Trends

### Customer Insights
- Average Ratings
- Purchase Patterns
- Customer Preferences

## Dashboard Visualizations

- KPI Cards
- Bar Charts
- Pie Charts
- Donut Charts
- Line Charts
- Funnel Charts
- Interactive Filters & Slicers

## Business Insights

- Supermarket Type 1 generates the highest revenue.
- Fruits, Vegetables, and Snack Foods are among the top-selling categories.
- Medium-sized outlets contribute significantly to overall sales.
- Customer ratings show consistent satisfaction across outlet types.
- Tier 3 locations demonstrate strong sales performance.
